# 개발진행 상황

## 엔진 프레임워크
1. 컴포넌트 기반으로 엔진 설계
2. 오브젝트의 계층 구조 설계
3. 시스템 매니저 구조 설계 (시간, 화면, 키이벤트, 충돌 등)

### 개발중
1. 오디오 컴포넌트
2. UI 프레임워크 및 클래스

#### 적용 기술
- 오브젝트의 회전 및 피봇 값 구현
- 스프라이트 애니메이션
- 2차원 Vector2 클래스 를 통한 좌표 및 충돌 계산
- 플레이어 주위를 이동하는 카메라
- 게임 오브젝트 관리
- 게임 오브젝트의 로컬좌표와 렌더링 좌표 분리
- 오브젝트의 회전은 항상 피봇좌표를 기준으로 회전 (XFORM 이미지 행렬계산, WinAPI)
